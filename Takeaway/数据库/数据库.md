# 数据库

所有的表都在这里

## 用户表

account

| 字段名      | 备注                       | 字段类型     | 约束       |
| :---------- | -------------------------- | ------------ | ---------- |
| userId      | 主键                       | INT(255)     | 主键，自增 |
| username    | 用户名（长度大于12小于20） | VARCHAR(20)  | 非空，唯一 |
| password    | 密码                       | VARCHAR(255) | 非空       |
| cname       | 名称                       | VARCHAR(20)  | 默认为空   |
| phone       | 手机号码                   | CHAR(11)     | 非空，唯一 |
| adress      | 地址                       | JSON         | 默认为空   |
| createdDate | 注册时间                   | DATETIME     | 非空       |

## 商户表

merchant

| 字段名       | 备注     | 字段类型     | 约束       |
| ------------ | -------- | ------------ | ---------- |
| merchantId   | 商户id   | INT(255)     | 主键，自增 |

## 订单表

order

| 字段名      | 备注     | 字段类型    | 约束                  |
| ----------- | -------- | ----------- | --------------------- |
| orderId     | 订单id   | INT(255)    | 主键，自增            |
| userId      | 用户id   | INT(255)    | 外键account约束       |
| merchantId  | 商户id   | INT(255)    | 非空                  |
| totalAmount | 总金额   | INT(255)    | 非空                  |
| status      | 状态     | VARCHAR(10) | 默认：’未完成‘        |
| createdDate | 创建时间 | DATETIME    | 非空                  |
| updatedDate | 更新时间 | DATETIME    | 默认：= ‘createdDate’ |
